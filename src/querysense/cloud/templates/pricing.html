Frequently Asked Questions

Is QuerySense open source?

Yes! QuerySense core is MIT-licensed. The CLI, analyzer, and all detection rules are free and open source. Cloud features (dashboard, team collaboration, compliance packs) are available in paid tiers.

Does QuerySense need access to my database?

No. Core analysis works entirely offline with EXPLAIN JSON output. Database connection is optional for Level 3 enhanced analysis (index validation, statistics freshness checks) and the watch daemon.

How does QuerySense compare to pganalyze?

pganalyze is a production monitoring dashboard ($149+/mo). QuerySense is a pre-production prevention tool (free-$99/mo). They're complementary: QuerySense catches issues before deployment, pganalyze monitors after. QuerySense uniquely offers CI/CD gating, plan regression detection, and compliance enforcement.

What PostgreSQL versions are supported?

QuerySense supports PostgreSQL 12-18. The upgrade validator has version-specific knowledge for PG14→18 optimizer changes. MySQL support is experimental.
